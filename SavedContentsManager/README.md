# Contents Manager

컨텐츠를 관리하는데 도움을 주는 프로그램

```
* 항목1
    * 001 항목1-1
        * file1.abc
        * file2.abc
    * 002 항목1-2
        * file1.abc
        * file2.abc
    * 003 항목1-3
        * file1.abc
        * file2.abc
* 항목2
    * 001 항목2-1
        * file1.abc
        * file2.abc
    * 002 항목2-2
        * file1.abc
        * file2.abc
```

귀찮으니 자세한 내용은 생략한다.

## Build

* Microsoft(R) Visual Studio Community 2015
* .Net Framework 4.5.2
* C#

## Installation

Windows:

```
Download exe file and execute it
```
