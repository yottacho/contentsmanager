# Contents Manager

시리즈 컨텐츠 관리 프로그램

개인적으로 써먹기 위한 유틸리티로 귀찮으니 자세한 내용은 생략한다.

## 주요 기능

* 컨텐츠 목록에서 세부 컨텐츠가 몇 개인지, 언제 마지막으로 업데이트 했는 지 보여주고 정렬
* 세부 컨텐츠에 대해 폴더별로 번호매기기 및 번호 재정렬
* 다른 폴더에 있는 컨텐츠를 관리 폴더로 병합하는 기능

* Microsoft(R) Visual Studio 2017
* .Net Framework 4.5.2
* C#
* SQLite nuget
